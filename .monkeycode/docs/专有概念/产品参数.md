# 产品参数 (ParameterRecord)

企业自有产品技术参数基准库中的数据记录，是投标比对的参照基准。

## 什么是产品参数？

产品参数代表企业自有产品能够满足的技术规格。作为产品参数基准库的核心数据，每条参数记录包含了产品在特定版本下的标称值、可满足范围、证明材料索引等信息，用于与招标参数进行逐项比对。

**关键特征**:
- 按产品线和版本分类管理
- 区分标称值与可满足范围
- 预置偏离类型用于快速匹配判定逻辑

## 结构

```python
@dataclass
class ParameterRecord:
    id: str
    version_id: str            # 所属产品版本
    name: str                  # 参数名称（标准名称）
    nominal_value: str         # 标称值
    acceptable_range: str      # 可满足范围
    unit: str                  # 单位
    deviation_preset: str      # 偏离类型预设（数值范围/枚举值/布尔型/功能描述）
    category: str              # 参数分类
```

## 版本管理

每个产品线可有多个 ProductVersion，每次产品迭代可创建新版本。分析任务选择一个版本，参数比对仅使用该版本的参数记录，不跨版本混合。
