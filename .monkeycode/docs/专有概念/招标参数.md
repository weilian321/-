# 招标参数 (ParsedParameter)

从招标文件中自动提取的结构化技术参数条目，是投标分析的基准输入。

## 什么是招标参数？

招标参数代表招标文件中对投标产品提出的单条技术要求。系统从 PDF/Word 招标文件中自动定位技术参数章节，提取每个参数及其属性，包括参数名称、要求值、单位、是否为实质性条款（*号标记）以及条款在原文中的位置。

**关键特征**:
- 自动从文档中提取，支持人工修正
- 标注是否为实质性条款（*号标记），不满足将直接导致废标
- 保留参数间的层级从属关系（parent_id/children）
- 记录原文位置以便溯源

## 结构

```python
@dataclass
class ParameterItem:
    id: str                    # 唯一标识
    name: str                  # 参数名称（如"防火墙吞吐量"）
    requirement_value: str     # 要求值（如"≥40Gbps"）
    unit: str                  # 单位（如"Gbps"）
    is_material: bool          # 是否为实质性条款（*号）
    param_type: ParamType      # 数值/枚举/布尔/功能描述
    source_location: str       # 原文位置（页码/段落）
    parent_id: Optional[str]   # 父级参数ID
    children: list[str]        # 子级参数ID列表
```

## 参数类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 数值范围 | 具有数值要求的参数 | 吞吐量≥40Gbps |
| 枚举值 | 在有限选项中选择 | 部署模式：反向代理/透明代理/旁路 |
| 布尔型 | 支持/不支持 | 支持IPv6 |
| 功能描述 | 文本描述的功能要求 | 支持智能威胁检测引擎 |
