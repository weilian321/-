# 技术得分

基于招标文件评分细则和参数偏离表计算的投标技术部分预估得分。

## 什么是技术得分？

技术得分是系统根据招标文件中的评分细则，将偏离表中的每条参数偏离结果映射到对应评分项，逐一计算得分后汇总得到的总分和得分率。用于预判投标竞争力。

## 评分类型

| 类型 | 说明 |
|------|------|
| 定量打分 | 参数满足预设条件即得固定分数 |
| 定性打分 | 根据优劣等级分级给分 |
| 加分规则 | 优于要求时额外加分 |
| 扣分规则 | 不满足要求时扣分 |

## 得分溯源

每项得分可追溯到:
- 评分标准原文
- 参数偏离结果
- 产品参数来源
- 证明材料索引
