# 参数偏离

招标参数与产品参数的符合性比对结果，是技术得分计算和风险判断的基础。

## 什么是参数偏离？

参数偏离表示投标产品参数相对于招标要求的差异状态。系统将招标参数与产品参数进行语义匹配后，按参数类型执行对应的判定逻辑，输出偏离类型和风险等级。

## 偏离类型

| 类型 | 含义 | 影响 |
|------|------|------|
| 正偏离 | 投标产品参数优于招标要求 | 加分项，竞争优势 |
| 无偏离 | 投标产品参数满足招标要求 | 正常得分 |
| 负偏离 | 投标产品参数低于招标要求 | 扣分，可能导致废标 |
| 无法确认 | 缺少证明材料或参数无法匹配 | 需人工确认 |

## 风险分级

- **废标级风险**: 实质性条款（*号标记）出现负偏离或无法确认
- **得分扣分项**: 普通参数出现负偏离
- **无风险**: 无偏离或正偏离
